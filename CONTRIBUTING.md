# Contributing

MemoryX uses a dual-license model:

- `AGPL-3.0-or-later` for the open-source public version.
- Separate commercial licensing by written agreement.

Because dual licensing requires clear rights ownership, external contributions
are not automatically accepted.

Before submitting substantial code, documentation, tests, or design changes,
contributors should expect to sign a CLA or another contributor agreement that
allows the project to keep offering both AGPL and commercial licenses.

Until that process is finalized, maintainers should treat unsolicited external
pull requests as review material only and should not merge them if doing so
would weaken the ability to dual-license MemoryX.
